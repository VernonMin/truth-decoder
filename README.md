# 职场黑话翻译站 (The Buzzword Decoder)

> 撕碎职场假面，还你人间清醒

一个极简的 H5 交互网页，用 AI 翻译职场黑话，揭露 PUA，提供情绪价值。

## 🎯 核心功能

- **AI 翻译引擎**：将职场黑话翻译成人话
- **PUA 等级评估**：1-5 星评级系统
- **老板心机拆解**：揭露话语背后的真实目的
- **神回怼**：提供高情商和直接发疯两种回复方式
- **爆火海报生成**：一键生成分享海报
- **黑客帝国风格**：黑黄配色，霓虹效果，扫描线动画

## 📝 AI 配置

### 使用真实 AI API

项目已经配置好 AI 提示词，位于 `src/config/aiPrompts.js`。

要使用真实的 AI API，需要：

1. 获取 Anthropic API Key
2. 在 `src/App.jsx` 中替换模拟数据调用为真实 API 调用：

```javascript
import { callAI } from './config/aiPrompts';

// 在 decodeJargon 函数中
const result = await callAI(text, YOUR_API_KEY);
```

### System Prompt

AI 被设定为一个在职场混迹 30 年、极度毒舌、看透资本本质的"职场教父"。

输出格式包括：
- 【翻译】：直白的人话翻译
- 【PUA 等级】：1-5 星评级
- 【老板心机】：揭露真实目的
- 【神回怼】：高情商和发疯两种回复

详细配置见 `src/config/aiPrompts.js`。
